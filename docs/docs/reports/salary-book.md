# Salary Book Register

The **Salary Book Register** is the master payroll register — one row per employee with every earning and deduction component as a separate column. Useful for audits, management review, and statutory submissions.

---

## What it shows

- Employee details (ID, name, department, designation).
- Every salary component as a dynamic column — components used in any slip in the filtered period appear automatically.
- Separate subtotals for earnings and deductions.
- Net pay per employee.

---

## Filters

| Filter | Description |
|---|---|
| Company | Mandatory |
| Month | Calendar month |
| Payroll Period | Financial year |
| Department | Optional drill-down |
| Employee | Optional single-employee view |

---

## Export

Export to Excel for submission to banks, auditors, or statutory authorities. The component columns dynamically match whatever components are in the salary slips — no manual configuration required.

---

## Use cases

| Use case | How |
|---|---|
| Salary audit | Export full register, verify totals against GL |
| Bank bulk upload | Use **Bank Mandate** report for net pay + IFSC per employee |
| Labour inspector submission | Print the register as a PDF |
| Cost allocation review | Filter by department, verify cost centre mapping |
